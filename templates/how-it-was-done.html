Making the model

A model represents what was learned by a machine learning algorithm. It is the "thing" that will be used to determine what an image represents, and thus also the very heart of the process.

When creating a model, we call it "training the model". We must teach our model how to calculate what an image represents. We do this by training it.
We did this with the help of so-called deep learning, and more specifically a convolutional neural network.
